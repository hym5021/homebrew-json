# Setup

```
brew tap nlohmann/json
```

# Install

```
brew install nlohmann_json
```

To install the bleeding-edge version:

```
brew install nlohmann_json --HEAD
```

# Test

```
brew test nlohmann_json
```

The formula needs to be installed before it can be tested.
